# Competitor Analysis: Aleno

## Overview
Aleno is an AI implementation provider for the gastronomy sector.

## Tech Stack
- **Cloud Infrastructure:** Primarily AWS (Amazon Web Services)
- **API Layer:** GraphQL (for fast, flexible data queries)
- **Backend:** Node.js or Python (standard for ML applications)
- **AI Models:** Custom ML models for table logic + LLMs (GPT series) for text
- **Frontend:** React / JavaScript (integrated into the HubSpot CMS of the website)

## Strategy Note
Positioned as a competitor for the Young Creatives gastronomy division (e.g., Lead Magnet "Unabhängigkeit von Portalen").
